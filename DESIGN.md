# Soundboard

A 4-screen drum-pad soundboard with 64 trigger buttons (16 per screen).
Core mechanic: tap any pad to play the matching sound file from the `sounds/` folder.
Art style: dark UI with vibrant coloured pads, no visual framework dependencies.

## Journal
- Initial build: 4 screens × 16 pads, nav bar at bottom, placeholder sound files in sounds/. Touch latency minimized via touchstart.
