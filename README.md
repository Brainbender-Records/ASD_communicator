The PDF and CSV sheets should be helpful in setting up a custom app with this. The csv can be used to help format the audio files in batch TTS or text to speech generators, use the name convention in the 2nd column. 

You will need to remove the Option 1 = in every cell before being able to use TTS because otherwise you will need to edit each audio file to remove half the recording. It is easy to forget this.

When you want to update hte button text remember I started at the top left of every screen, so top left to right goes 1,2,3,4 then next row 5,6,7,8, next row 9,10,11,12, last row 13,14,15,16.
Each sheet follow this same format, and all the sheets are listed in the doc. The list will help organize what screen has what text.
